# House-Sales-Dashboard

## 📌 Project Overview
Using an interactive Tableau dashboard, this project analyzes house sales in Kings County, Seattle Washington.

## 📊 Features
- Line Graphs
- Histograms
- Heat Maps
- Map
- Slicers

## ⚙️ Technologies Used
